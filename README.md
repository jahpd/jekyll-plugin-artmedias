# Jekkyl-media-plugins

These are personal plugins i created to use in my projects:

  - Youtube

```liquid
{% youtube EoTCS9AZRSQ %}
```

  - <video> Tags

```liquid
{% video /myvideo.mp4 %}
```

  - MIDI with <embed> tags

```liquid
{% midi /mymidi.mid %}
```

  - Vexflow <canvas> tags (Still in developing) for create music staffs

```liquid
{% vexflow myflow 400 170 %}
tabstave
{% endvexflow %}
```
